# La Matemática del Trading

Página web educativa sobre los fundamentos matemáticos aplicados al trading.

## Contenido

- Gestión del riesgo y el capital
- Probabilidad y esperanza matemática
- Análisis estadístico de operaciones
- Sistemas de posicionamiento

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Cómo ejecutar

Abre `index.html` en tu navegador.
